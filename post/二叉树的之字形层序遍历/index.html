二叉树的之字形层序遍历

2021-05-04 17:24
tree
约 407 字 预计阅读 1 分钟 次阅读

题目描述

给定一个二叉树，返回该二叉树的之字形层序遍历，（第一层从左向右，下一层从右向左，一直这样交替） 例如： 给定的二叉树是{3,9,20,#,#,15,7},

该二叉树之字形层序遍历的结果是

[

[3],

[20,9],

[15,7]

]

示例1

输入

1

{1,#,2}

返回值

1

[[1],[2]]

答案

本题的解题的关键是定义一个 i 值，这个值来代表是奇数层还是偶数层，如果是偶数层，则按照正常的顺序添加，如果是奇数层，那么每次都添加到 列表的第一个位置，从而实现之字形遍历

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 * }
 */
import java.util.Stack;
public class Solution {
    /**
     * 
     * @param root TreeNode类 
     * @return int整型ArrayList<ArrayList<>>
     */
    public ArrayList<ArrayList<Integer>> zigzagLevelOrder (TreeNode root) {

       ArrayList<ArrayList<Integer>>  res = new ArrayList<>();
       Queue<TreeNode> s = new LinkedList<>();
        if(root != null){
            s.add(root);
        }     
        int i = 0;
        while(!s.isEmpty()){
            int ns = s.size();
            ArrayList<Integer> r = new ArrayList<>();
            for(int j=0; j < ns; j++){
                TreeNode node = s.poll();
                if( i % 2 == 0){
                    r.add(node.val);
                }else{
                    r.add(0, node.val);
                }
                if(node.left != null){
                    s.offer(node.left);
                }
                if(node.right != null){
                    s.offer(node.right);
                }               
            }
            res.add(r);
            i ++;
        }
        return res;
    }
    
}
