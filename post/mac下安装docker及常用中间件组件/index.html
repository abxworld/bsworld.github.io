一、mac下安装docker

1、命令行安装

1

brew cask install docker

2、手动安装

docker download

二、docker 安装中间件组件

1、安装redis

  • 查找redis的docker镜像

    1

    docker search redis

  • 拉取镜像到服务器本地

    1

    docker pull docker.io/redis

  • 运行镜像生成redis容器

    1

    docker run -p 6379:6379 --name redis -d redis:latest redis-server

  • 查看启动的容器

    1

    docker ps

  • 进入容器

    1

    docker exec -it container_id redis-cli

2、安装zookeeper

  • 拉取zookeeper最新的镜像

    1

    docker pull zookeeper

  • 运行zk镜像，并命名

    1

    docker  run  -p 2181:2181 --name zk -d zookeeper:latest

  • 查看启动的容器

    1

    docker ps

  • 进入容器

    1

    docker exec -it container_id bash

3、安装mysql

  • 拉取mysql 5.7的镜像

    1

    docker pull mysql:5.7

  • 从镜像创建mysql 5.7容器

    1

    docker run --name ms -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.7

  • 查看已经运行的容器

    1

    docker ps

  • mysql容器内相关操作

    • 进入mysql 容器

      1

      docker exec -it container_id bash

    • 进入mysql 客户端

      1

      docker exec -it ms mysql -uroot -p123456

    如果在连接数据库的时候报 access denied 错误 在mysql客户端执行以下命令:

    GRANT ALL PRIVILEGES ON . TO’root’@'%’ IDENTIFIED BY ‘123456’ WITH GRANT OPTION;

三、启动客户端命令别名设置

  1. 配置所在的文件名:

    ~/.bash_profile

  2. 配置命令内容

    • redis

      1

      alias redis-cli='docker exec -it redis redis-cli'

    • zookeeper

      1

      alias zk-cli='docker exec -it zk ./bin/zkCli.sh'

    • mysql

      1

      alias ms-cli='docker exec -it ms mysql -uroot -p123456'

  3. 配置完成后需要使用命令,使别名生效

    1

    source ~/.bash_profile

  4. 解决新打开的终端,别名失效的问题

    1. 编辑~/.zshrc文件

      1

      vim ~/.zshrc

    2. 在~/.zshrc加入以下命令

      1

      source ~/.bash_profile

四、docker容器导出与导入

  1. export、import

    • 列出所有的容器

      1

      docker container ls -a

    • 容器导出

      1

      docker export contaier_id > self_define.tar

    • 容器导入成镜像(images)

      1

      docker import self_define.tar self_define_image_name

    容器导出成tar包，然后tar包导入成镜像，运行镜像即可生成新的容器，新的容器不会保留老容器的数据

五、docker 常用命令

  1. 启动docker服务

    mac 环境:

    1

    open /Applications/Docker.app

    Linux环境:

    1

    service docker start

  2. 关闭docker 服务

    mac 环境:

    1

    launchctl list | grep 'com.docker.docker' | awk '{print $3}' | xargs launchctl stop

    Linux环境:

    1

    serive docker stop

  3. 查看正在运行的docker 容器

    1

    docker ps

  4. 列出所有的container

    1

    docker ps -a

  5. 列出最近一次启动的container

    1

    docker ps -l

  6. 启动docker 容器

    1

    docker start container_id

  7. 关闭docker容器

    1

    docker stop container_id

  8. 重启容器

    1

    docker restart container_id/container_name

  9. 重命名容器名称

    1

    docker rename contaier_id/container_name new_name

  10. 列出本地所有的镜像

1

docker images

  1. 删除指定的镜像id

    1

    docker rmi images_id

  2. 删除容器

    1

    docker rm container_id

  3. 查看docker系统的信息

    1

    docker info

  4. 查看docker版本

    1

    docker version

  5. 重命名容器名称

    1

    docker rename old_container_name new_container_name

  6. 可以查看到容器主程序的输出

    1

    docker logs container_id/container_name
