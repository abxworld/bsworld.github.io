一、背景

​ 提供一个统一的配置客户端进行配置信息的查询及缓存,简化使用方式

  • 内置日志更新组件(之前的ZkLogger)
  • 内置连接zookeeper域名解析组件(之前的ZKInitUtil的getUrlByIndex()方法)
  • 统一组件到统一个配置bean
    • ZooKeeperCacheOperations（操作zookeer的组件及缓存的组件）
    • ZkConfigurationDynamic(对ZooKeeperCacheOperations的封装)
    • ZkDynamicKeyEnum(zk节点地址枚举)

@LBClass(groupName = "/lb")
@Component
public class Student {
    private Integer id;

    private String name;

    private String logPath;

    private boolean young;

    @LBItem(key = "/student/id", value = "10")
    public Integer getId() {
        return id;
    }

    @LBItem(key = "/student/young", value = "false")
    public boolean getYoung() {
        return young;
    }

    @LBItem(key = "/student/name", value = "hello_world")
    public String getName() {
        return name;
    }

    @LBItem(key = "/student/logPath", value = "info", isLogPath = true)
    public String getLogPath() {
        return logPath;
    }
}

@LBClass:

作用:作用在类上,标识这是一个配置类，可以将将所有的配置统一写在这里，客户端配置系统会扫描配置的包中标有该注释的类。

groupName:分组名称，改配置类的最顶级节点

enableGray:灰度标识，默认false不启动灰度

@LBItem

作用:作用在方法上，标识获取具体配置的方法，方便进行代理

key:具体的配置节点

value:默认值

associateField:关联的属性名称，如果该方法是ide自动生成的,以get或者is开头，可以不写

isLogPath:标识该节点是否是日志节点，若是，在该节点变更的时候会同步变更日志级别

三、流程设计

1、系统设计原理

  • zookeeer作为中心节点存储数据

  • 管理员增删数据，修改配置

  • 客户端监听zookeeper节点，更新本地缓存

2、系统设计简单流程

  • 通过扫描组件扫描配置类（@LBCLASS注解的类）,清洗配置类数据,生成一个节点到配置类信息的映射的map(之所以不用spring相关组件去获取，是为了与spring解耦，不依赖于spring)

  • 生成配置类的代理(配置信息的缓存)，并将代理注册到spring容器中,这样在引用配置bean的时候，会先经过这个代理获取配置信息(之所以不用aop，一方面是为了与spring解耦，一方面是能少配置一个aop切面,简化使用方式)

  • 根据扫描到的信息，以分组名为节点注册watcher到zookeeper并进行监听

  • zookeeper信息更新时，根据事件类型的不同(创建节点，更新节点，删除节点)，派发到不同的事件处理器中，更新缓存或者置缓存为不可用

3、代码分享

​ 讲一下bean的动态代理及spring bean的动态注入

​ 所谓的动态注入,本质上就是将已有的对象注入到spring容器，这样就可以对注入的bean进行aop切面，ioc等操作

举例:

​ mybatis中的mapper接口,是由myabtis搜集到相关的mapper，并对mapper接口生成相应的代理类，并注入到spring容器中，这样我们才能用引用相关的mapper bean

  • ​ bean的动态注入实现类
    • 实现BeanDefinitionRegistryPostProcessor接口,获取Registry(bean注册器),构造GenericBeanDefinition(bean定义信息)对象,注入spring bean definition容器
    • AbstractStarter是一个模板类，在start()方法内会调用before()方法,进行bean的注册

public class SpringStarter extends AbstractStarter implements DisposableBean, BeanDefinitionRegistryPostProcessor, PriorityOrdered {
    private static AtomicInteger ai = new AtomicInteger(1);
    private static final Logger logger = LoggerFactory.getLogger(SpringStarter.class);
    private BeanDefinitionRegistry registry;
    /**
     * 释放资源
     */
    @Override
    public void destroy() {
        super.destroyCore();
    }

    @Override
    public void postProcessBeanDefinitionRegistry(BeanDefinitionRegistry registry) throws BeansException {
        this.registry = registry;
        logger.info("postProcessBeanDefinitionRegistry(), invoke times:{}",ai.getAndIncrement());
        this.start();
    }

    @Override
    public void postProcessBeanFactory(ConfigurableListableBeanFactory beanFactory) throws BeansException {

    }

    @Override
    public int getOrder() {
        return Ordered.HIGHEST_PRECEDENCE + 1;
    }

    @Override
    protected void before() {
        ScanStaticModel scanStaticModel = this.scanStaticModel;
        BeanDefinitionRegistry registry = this.registry;
        Set<Class<?>> clazzList = scanStaticModel.getClazzList();
        BeanHolder<Object> holder = BeanHolder.getHolder();
        for (Class<?> clazz : clazzList) {
            Object originBean = holder.getBean(clazz);
            GenericBeanDefinition beanDefinition = new GenericBeanDefinition();
          //指定bean类型 
        	  beanDefinition.setBeanClass(SpringFactoryBean.class);
  	 // 向工厂bean的构造方法里注入原始bean的信息，在获取配置信息的时候用的到    		      
          beanDefinition.getConstructorArgumentValues().addGenericArgumentValue(originBean);
            beanDefinition.setLazyInit(false);
            beanDefinition.setAbstract(false);
            beanDefinition.setAutowireCandidate(true);
            beanDefinition.setScope("singleton");
            String simpleName = clazz.getSimpleName();
            String preName = simpleName.substring(0, 1).toLowerCase();
            String lastName = simpleName.substring(1);
            simpleName =  preName + lastName;
          //注册bean信息
            registry.registerBeanDefinition(simpleName, beanDefinition);
        }
    }
}

 public void start() {
        try {
            if (initFlag.compareAndSet(StateEnum.UNSTART, StateEnum.STARTING)) {
              //扫描配置类信息，进行数据的清洗
                scan();
              //对扫描的配置类信息,生成代理，并注入BeanHolder容器，
              registerBean();
              //生成spring的注册对象，并通过相应的注册器注册对象到spring容器中
                before();
                startCore();
                after();
                initFlag.set(StateEnum.STARTED);
                logger.info("LB-BS-CLIENT start success");
            }
        } catch (Throwable t) {
            initFlag.set(StateEnum.CLOSE);
            logger.error("start（）, error:", t);
            throw t;
        }
    }

//只做了一件事就是对配置bean生成代理并注入BeanHolder容器中

private void registerBean() {
        ScanStaticModel scanStaticModel = this.scanStaticModel;
        Set<Class<?>> clazzList = scanStaticModel.getClazzList();
        BeanHolder holder = BeanHolder.getHolder();
        for (Class<?> clazz : clazzList) {
            try {
                Object originBean = clazz.newInstance();
                holder.registerBean(clazz, originBean);
                Object proxy = LBProxyFactory.getProxy(clazz, originBean);
                holder.registerProxy(clazz, proxy);
                holder.registerLbClassBean(originBean);
            } catch (Exception e) {
                logger.error("lb-bs-client bean register error:", e);
                throw new IllegalStateException("lb-bs-client bean register error", e);
            }
        }
        }
    }

public class SpringFactoryBean<T> implements FactoryBean<T> {
    Object originBean;
  
    @Override
    public T getObject() throws Exception {
        Object proxyBean = BeanHolder.getHolder().getProxyBeanByOriginClazzType(originBean.getClass());
        return (T) proxyBean;
    }

    @Override
    public Class<?> getObjectType() {
        return BeanHolder.getHolder().getProxyBeanByOriginClazzType(originBean.getClass()).getClass();
    }

    @Override
    public boolean isSingleton() {
        return true;
    }

    public SpringFactoryBean(Object originBean) {
        this.originBean = originBean;
    }
}

public class LBProxyFactory {

    public static <T> T getProxy(Class<T> clazz, Object originBean) {

        T proxy = null;
        try {
            proxy = create(clazz, originBean);
        } catch (Exception e) {
        }

        return proxy;
    }

    private static <T> T create(Class<T> clazz, Object originBean) throws IllegalAccessException, InstantiationException {
        Enhancer enhancer = new Enhancer();
        enhancer.setSuperclass(clazz);
        enhancer.setCallback(new LbInterceptor(originBean));
        T t = (T) enhancer.create();
        return t;
    }
}
  • 代理切面类:
    • 对于方法上上没有标记@LBItem注解的认为是非配置节点，直接走原生方法调用，如toString()等

public class LbInterceptor implements MethodInterceptor {
    private static final Logger logger = LoggerFactory.getLogger(LbInterceptor.class);

    private Object originBean;

    public LbInterceptor(Object originBean) {
        this.originBean = originBean;
    }

    @Override
    public Object intercept(Object o, Method method, Object[] objects, MethodProxy methodProxy) throws Throwable {
        Object result = null;
        String methodName = null;
        try {
            LBItem lbItemAnno = method.getDeclaredAnnotation(LBItem.class);
            if (lbItemAnno == null) {
                result = methodProxy.invokeSuper(o, objects);
                return result;
            }
        	//1、....从zookeeper节点信息对应的本地缓存里获取配置信息
            Object cacheVal = LBCache.getCacheVal(method.getDeclaringClass(), lbItemAnno);
            if (cacheVal != null) {
                return cacheVal;
            }
            if (cacheVal == null) {
                try {
                    //2、从配置类的默认值里获取配置信息
                    methodName = method.getName();
                    Method originMethod = originBean.getClass().getDeclaredMethod(methodName);
                    if (originMethod != null) {
                        result = originMethod.invoke(originBean, objects);
                    }
                } catch (Throwable e) {
                    logger.error("LbInterceptor.class,methodName:{}, e:", methodName, e);
                }
                return result;
            }

        } finally {

        }
        return result;

    }
}
